# Task Manager MERN Application

A clean, production-ready full-stack task management application engineered using the MERN ecosystem. This project features a completely decoupled architecture, combining a robust Node.js/Express.js REST API backend with a responsive, modern React.js frontend.

## 🚀 Key Features

* **Full CRUD Functionality:** Seamlessly create, read, update, and delete tasks dynamically.
* **Decoupled Architecture:** Clean separation of concerns between client-side rendering and server-side business logic.
* **Secure Environment Architecture:** Sensitive cloud database connection strings and credentials are fully isolated using environment variables (`.env`).
* **Rigorous API Testing:** All backend RESTful endpoints have been comprehensively verified using Thunder Client.

---

## 🛠️ Tech Stack

### Frontend
* **Library:** React.js
* **Languages:** JavaScript (ES6+), HTML5, CSS3

### Backend
* **Runtime Environment:** Node.js
* **Framework:** Express.js
* **Database:** MongoDB Atlas (Cloud Cluster)
